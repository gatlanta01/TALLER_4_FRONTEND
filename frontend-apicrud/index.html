Acerca del Sistema

GestiónPro es un sistema de administración de productos electrónicos construido con Bootstrap 4, jQuery y una API REST en PHP.

Permite realizar operaciones CRUD completas: Crear, Leer, Actualizar y Eliminar productos del catálogo de manera segura y eficiente.
